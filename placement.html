Installation

Hebergement d'un site web sur une machine virtuelle

Serveur apache
Commande :

sudo apt install apache2

Si ça ne marche pas comme ici, faire :
Commande :

sudo apt-get update

Et refaire
Commande :

ls /var/www/html

Verifier si ça marche bien
Commande :

ls –l /var/www/html

Créer un repertoire et faire un test
Commande :

mkdir montuto.fr
cd montuto.fr
echo “ Votre message test “ > test.html

Aller dans le repertoire d’apache2
Commande :

cd /etc/apache2/
Puis dans sites-enabled
cd sites-enabled


Commande :

sudo vi 000-default.conf
Et se déplacer avec les flèches


Commande :

:wp
Pour enregistrer

Maintenant il faut rafraichir le systeme:
Commande :

sudo systemctl reload apache2.service


Commande :

cd..
