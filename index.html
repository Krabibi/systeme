Introduction à la machine virtuelle

Une machine virtuelle est un environnement informatique isolé qui simule les ressources d’un ordinateur physique, permettant l’exécution d’un système d’exploitation et d’applications. Elle est composée de fichiers de spécification et de configuration et est soutenue par les ressources physiques d’un hôte.

Comment installer une machine virtuelle
